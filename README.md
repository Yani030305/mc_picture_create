# Minecraft 食物贴图转换器

将食物图片转换为 Minecraft Mod 可用的 64×64 物品贴图。

## 功能特性

- 图片上传（点击或拖拽）
- 自动背景去除（flood fill 算法）
- 自动裁剪主体
- 等比缩放至 64×64
- nearest-neighbor 最近邻缩放（保持像素清晰）
- 色彩量化（16/24/32/64 色）
- PNG 透明背景输出
- 下载文件名：`原文件名_64x64.png`

## 使用方法

1. 用浏览器打开 `index.html`
2. 上传食物图片
3. 调整参数（如需要）
4. 点击「下载贴图」

## 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 背景去除阈值 | 235 | 0-255，值越高越容易去除浅色背景 |
| 饱和度阈值 | 30% | 低于此饱和度的浅色像素会被视为背景 |
| 安全边距 | 3px | 缩放后保留的边距 |
| 色彩量化 | 开启 | 开启后启用色彩量化 |
| 色彩数量 | 32 | 量化后的颜色数量 |

## 输出规格

- 文件尺寸：64×64 像素
- 格式：PNG
- 背景：真透明（alpha=0）
- 缩放算法：nearest-neighbor（无抗锯齿）

## 放置位置

下载的 PNG 文件放入 Mod 资源目录：

```
src/main/resources/assets/你的modid/textures/item/
```

## 技术实现

- 纯 HTML + CSS + JavaScript（无后端）
- Canvas API 处理图片
- flood fill 算法去除背景
- RGB 分桶量化（无抖动）
